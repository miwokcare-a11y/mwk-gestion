Configuración inicial
Para sincronizar en todos tus dispositivos necesitás crear una base de datos gratuita en Supabase. Solo se hace una vez.
1
Entrá a supabase.com y creá una cuenta gratuita. Luego creá un New Project.
2
En tu proyecto, andá a SQL Editor y ejecutá el script de creación de tablas que está más abajo (copiá y pegá).
3
Andá a Settings → API. Copiá la Project URL y la Publishable key y pegálos acá abajo.
